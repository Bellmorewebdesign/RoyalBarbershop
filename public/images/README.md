# Shop photos go here

Drop the **real Royal Barber Shop photos** into this folder using the exact
filenames below. Each one is already wired into the site. Until a file exists,
the site shows a tasteful branded fallback panel (no broken images), so nothing
looks unfinished — but the real photos make it shine.

> Only use the shop's own photos here. No stock photos, AI photos or external
> image URLs (per the project brief).

| Filename                | Where it appears            | Suggested photo (from the ones you sent)                          |
| ----------------------- | --------------------------- | ----------------------------------------------------------------- |
| `storefront-night.jpg`  | Hero background             | The lit-up blue "Royal BARBER SHOP" sign at night                 |
| `barber-kid-car.jpg`    | About — main image          | Barber cutting a child's hair in the Lightning-McQueen car chair  |
| `interior-boy.jpg`      | About — small overlap image | Inside shot: boy in orange shirt, barber working in the back      |
| `street-view.jpg`       | Visit / Location card       | Street view of the storefront (green awning, "BARBER SHOP" sign)  |
| `storefront-day.jpg`    | (optional / spare)          | Daytime storefront with the teal awning and brick building        |

## How to swap or rename

- Keep the filenames above **or** edit them in `src/lib/images.js`.
- Recommended: JPG or WebP, roughly 1600px on the long edge, optimized for web.
- After adding files, just refresh the dev server — no code changes needed.
