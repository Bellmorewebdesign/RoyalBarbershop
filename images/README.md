# Images

These are the shop's own photos and logo, used across the site. To replace one,
drop in a new file with the **same name** (or update the `src="./images/..."`
reference in `../index.html`).

| File                   | Used for                          |
| ---------------------- | --------------------------------- |
| `Transparent_logo.png` | Logo in the nav, hero and footer  |
| `normal_logo.png`      | Logo on a white background (spare) |
| `storefront-day.png`   | Main hero photo                   |
| `barber-kid-car.png`   | Small inset photo in the hero     |
| `interior-boy.png`     | About section photo               |
| `storefront-night.png` | "Walk right in" call-to-action band |
| `street-view.png`      | Visit / Location photo (links to Google Maps) |

Only the shop's own images are used here — no stock photos, AI photos or
external image URLs.
